# pda_printer

PDA Printer is a flutter project that uses the PDA 3505 SDK to print/scan. The project uses interaction with native code (Android only) to do the following:
- Get battery level
- Get charging status
- Print text/images/qr/barcode etc
- Scan qr/bar code

## Requirement
- Android version >=6
